英文名：[RubicCube]
喵语言：[lobitus]
中文名：[洛比特斯]

[游戏梗概]
lobitus是一个有6个面的奇怪立方体（立方体不都是有6个面的吗），但是这6个面是可以旋转的，用手指在6个面上搓动，立方体的面就会转起来。

[玩家指引]
随着星际尘埃的聚集，星际间引力场发生了非常大的变化，引力场方向变得极不稳定，lobitus的其中5个面在一次意外中发生了失控的旋转。从古至今一直流传着一个关于lobitus的传说，lobitus自从在我们星球中心发现之后，就一直维持着我们星球表面的稳定，万物矗立而不倒，精气传流而不息。而如今lobitus已经异位，在下次引力场张动回归之时，定会世界大乱。

很明显，复原的重任就交给玩家你了，我们已经为你准备好了远程虚拟控制接口，在一个透明的板前就能操控lobitus。进入游戏后，可以进行离线模式，或者在线模式。离线模式下，只能进行单机复原。在线模式下，玩家还可以进行挑战模式，挑战模式初始为一个固定复杂度的lobitus，若玩家完成游戏的时间超过挑战榜上的时间，则玩家可以更新挑战榜上对应条目，并获得一定积分，英雄榜可以查看每个玩家的总积分。

[管理员指引]
lobitus内部组织已经达成一致，他们是不会告诉我们任何有关lobitus更多的内幕的，但是我们有一个服务端，且只有一个，在服务端的文件夹里，README.txt文件透露了很重要的信息，不妨看看。

[测试人员指引]
请测试人员放心，lobitus已经经过特殊虚拟接口非对称保护处理，意外的lobitus相位是不会影响我们的安全的，但是请一定要阻止用力敲打透明板的玩家。lobitus比较特殊，并没有额外的测试接口可以使用。

[彩蛋]
lobitus很早之前就设置了若干彩蛋，3个或者是4个之类的，很少人能发现。

[lobitus 透明板操作说明]
经过我们对lobitus 接口特殊化处理后，一个手指可以完成全部操作，在灰色区域滑动手指，可以观察lobitus各个面和细节，将手指贴在lobitus表面，搓动透明板即可让lobitus跟随转动，放手后，lobitus会自动恢复到最稳定位置，如果在恢复到稳定位置之前，再次用手指触摸透明板lobitus的表面，可以将该面停下来。最后，要是出现比较混乱的情况，可以找到重新开始的符文，重新进行操作。
